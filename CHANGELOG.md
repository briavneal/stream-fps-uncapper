# CHANGELOG.md

## [Uncapper v2.4.1] - 2023-10-01
### Fixed
- Resolved a crash issue when enabling uncapping during gameplay in Split and Haven maps.
- Fixed improper display of FPS in the settings overlay when using a custom resolution.
  
### Changed
- Optimized the algorithm that calculates frame rates, resulting in smoother gameplay performance.
- Updated the UI for the FPS display to align with Valorant’s aesthetic.

### Added
- Introduced a new hotkey feature to enable/disable uncapping while in-game, enhancing user experience.
- Added support for recent Nvidia and AMD graphics cards to enhance compatibility.

## [Uncapper v2.3.0] - 2023-08-10
### Fixed
- Corrected an issue where the installed status of Riot Vanguard was not properly recognized by the app.
- Addressed a bug where FPS settings would reset after an unexpected game update.

### Changed
- Improved detection of Valorant's dynamic resolution scaling to ensure uncapping works smoothly regardless of changes.
- Minor changes to the loading screen graphics for better visibility during startup.

### Added
- Added tooltips for better guidance in the settings panel, particularly for the "Advanced Options".
- Implemented a feedback option to allow users to report issues directly through the app.

## [Uncapper v2.1.0] - 2023-06-15
### Fixed
- Resolved compatibility issues with Windows 11 that caused crashes on launch.
- Fixed a graphical glitch when uncapping FPS while using high refresh rate monitors.

### Changed
- Refined the method for retrieving screen refresh rates, improving accuracy on adaptive sync monitors.
- Enhanced the application’s launch speed by optimizing initialization processes.

### Added
- Introduced a “Performance Mode” toggle that optimizes system resources when Valorant is active.
- Added settings to adjust the uncapped FPS limit per game mode, supporting both Competitive and Unrated modes.

## [Uncapper v2.0.0] - 2023-04-05
### Added
- Initial release of Valorant FPS Uncapper, providing the ability to uncap frame rates beyond the standard 144 FPS limit.
- Basic UI interface for setting FPS limits and toggling uncapping on and off.
- Support for automatic updates to ensure users have the latest features and fixes.
